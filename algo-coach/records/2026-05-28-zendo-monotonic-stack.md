# 单调栈 — 海龟汤 (2026-05-28)

模式: 海龟汤 (是/不是/是也不是)
内容: 单调栈的通用骨架

## 提问记录

| 轮次 | 提问 | 判定 |
|------|------|------|
| 1 | 入栈决定左边界，出栈决定右边界 | ✔ 是 |
| 2 | 栈内的元素始终单调 | ✔ 是 |
| 3 | 目标值一般只和栈顶元素相关 | ✘ 不是 |
| 4 | 栈顶元素通常只是比较对象 | ✔ 是 |
| 5 | 计算通常发生在出栈的时候 | ✔ 是 |
| 6 | 栈里的元素不一定会出完 | ✔ 是 |
| 7 | 出栈计算 = Δh × Δw, 宽通常=1 | ⚡ 是也不是 |
| 8 | 宽跨多列, 每列宽1, i-left-1 根 | ✔ 是 |
| 9 | 综合: 二维约束空间+积压+单调 | ⚡ 是也不是 |
| 9' | 二维 = 下标索引 + 值, 可画图 | ✔ 是 |

## 最终骨架

| 维度 | 结论 |
|------|------|
| 触发时机 | 出栈时计算，左右边界都找到才动手 |
| 栈的角色 | 积压—暂存还没找右边界的候选 |
| 数据结构 | 二维: 下标(宽度) + 值(高度) |
| 计算模式 | Δh × Δw, 跨多列 |
| 单调性 | 保证新元素一次性连锁释放同类待定项 |
| 残留处理 | 遍历完栈里未出栈的通常不参与计算(或补算) |

结果: 通关 (7✔ 1✘ 2⚡, 共10轮)
