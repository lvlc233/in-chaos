# 问题结构 → 算法范式映射

归档: 2026-05-28 | 来源: 接雨水(42) / 最大矩形(84) 推导过程中总结

## 核心原则

问题的可剥削结构类型决定了最简可用算法的上界。

| 题目具有的性质 | 适用算法 | 典型复杂度 |
|---|---|---|
| 瓶颈单调触发 | 单调栈 | O(n) |
| 单调消去规则 | 双指针 | O(n) |
| 贪心选择性质 | 贪心 | O(n) / O(n log n) |
| 窗口合法条件单调 | 滑动窗口 | O(n) |
| 搜索空间单调有序 | 二分 | O(log n) |
| 问题可独立切割 | 分治 | O(n log n) |
| 最优子结构 + 重叠子问题 | DP | O(n^k) |
| 区间查询可结合 | 前缀和/树状数组/线段树 | O(1)/O(log n) |
| 只有合并没有拆分 | 并查集 | O(α(n)) |
| 图无环 + 偏序关系 | 拓扑排序 | O(V+E) |
| 边权非负 | Dijkstra | O(E log V) |
| 子问题对称可折叠 | 折半搜索 | O(2^(n/2)) |
| n ≤ 20 | 状压 DP | O(n·2ⁿ) |
| 流网络有对偶 | 最大流/最小割 | O(VE²) |
| 都没有 | 暴力穷举 | O(2ⁿ) |

## 关键洞察

1. **不是"算法选问题"，是"问题筛选算法"** — 题目有哪些性质，决定了哪些范式可用
2. **贪心是 DP 的子集** — 贪心能解的 DP 也能，但更慢；贪心需要更强的额外性质
3. **层次关系**：多个性质同时满足 → 多种算法可选，取最简

## 接雨水系列四题的底层结构

42/84/85/739/11 共有的核心性质：**瓶颈单调触发** — 一个更小的（或更大的）新元素同时"截断"一批待定项。
