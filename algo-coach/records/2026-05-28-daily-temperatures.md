# 739. 每日温度

日期: 2026-05-28 | 状态: 已阅读

## 结构性质

**瓶颈单调触发 → 单调栈（递减栈）**

找下一个更大的元素 (Next Greater Element)，单调栈最纯净形态。

## 逐步推演

```python
def dailyTemperatures(t):
    stack, res = [], [0]*len(t)
    for i, v in enumerate(t):
        while stack and v > t[stack[-1]]:
            pre = stack.pop()
            res[pre] = i - pre
        stack.append(i)
    return res
```

输入: `[73, 74, 75, 71, 69, 72, 76, 73]`
输出: `[1, 1, 4, 2, 1, 1, 0, 0]`

```
Step 0: i=0, v=73  栈:[73(░)] 入栈, 等待更高
Step 1: i=1, v=74  74>73 → pop 73, res[0]=1, 74入栈  栈:[74(░)]
Step 2: i=2, v=75  75>74 → pop 74, res[1]=1, 75入栈  栈:[75(░)]
Step 3: i=3, v=71  71<75, 不触发  栈:[75(░),71(░)]
Step 4: i=4, v=69  69<71, 不触发  栈:[75(░),71(░),69(░)]
Step 5: i=5, v=72  72>69 → pop 69, res[4]=1
                   72>71 → pop 71, res[3]=2
                   72<75, 停止, 72入栈  栈:[75(░),72(░)]
Step 6: i=6, v=76  76>72 → pop 72, res[5]=1
                   76>75 → pop 75, res[2]=4
                   栈空, 76入栈  栈:[76(░)]
Step 7: i=7, v=73  73<76, 不触发  栈:[76(░),73(░)]
                   遍历结束, 剩余元素保持 0

结果: [1, 1, 4, 2, 1, 1, 0, 0]
```

## 关键观察

- 72 一次触发了两个待定项 (69, 71) — 连锁释放
- 76 一次清空了整个栈 — 全局瓶颈
- 栈始终递减: 新元素小的入栈等, 大的往上消
- 每个元素进一次出一次: O(n)
