# 环形链表 II (Linked List Cycle II)

LeetCode 142 | 归档: 2026-06-08 | 状态: 未解决

## 追问阶段卡点

1. **从相遇点到入口的推导受阻**：能够模拟快慢相遇，但对"相遇后同速从头和从相遇点出发，重合点恰为环入口"的数学原理未能内化
2. **核心障碍**：头到入口的距离 与 相遇点沿环走回入口的距离 总是相等——用户无法从直觉上接受这不是巧合而是必然
3. 尝试了"补缺口"脚手架，未突破

## 核心待解

从等式 `fast = 2 × slow` 中：fast = a + n·r + k, slow = a + m·r + k，推导出 a = (n-2m)·r - k，即 a ≡ -k (mod r)，也即 a + k ≡ 0 (mod r)。相遇后从头走 a 步到入口，从相遇点走 a 步则正好偏移 a+k ≡ 0 (mod r)，恰好停在入口。

需要在下一轮学习中用自己的话重新推导这个环节。
