# 三数之和 (3Sum)

LeetCode 15 | 归档: 2026-06-08 | 状态: 待复习

## 追问阶段卡点

1. **内层循环终止条件**：最初认为"=0 就停"，用 `[-2,-1,0,1,2,3]` 固定位 `-2` 做脚手架发现 `[-2,0,2]` 会漏——正确条件是 `L < R`
2. **剪枝边界**：`≥0 就停` → `[0,0,0]` 会漏，更正为 `> 0`
3. **去重三层**：固定位跳过同值、L 跳过同值、R 跳过同值——由两个连续 `-1` 固定位的例子推导出

## 核心洞察

N 数之和 → 排序 + 固定 N-2 个 + 双指针降维扫描。单调性保证指针移动方向可预测，解后不终止继续搜，去重是易漏点。

## 用户总结提炼

双指针特征：数据单调 + 对偶性，通过拆解原问题降维为两端拟合。找到目标后不结束循环，需处理去重和终止边界。
